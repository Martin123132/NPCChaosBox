Known Limitations

Windows-first public alpha. Python 3.10 or newer is still required.

No accounts, API keys, cloud calls, Ollama, npm, or installer.

NPCs are deterministic. Some chaos is funny, some needs seed tuning.

Session packs, share cards, and packaged EXE builds are later steps.

Feedback

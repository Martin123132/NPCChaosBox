## Page Copy

NPC Chaos Box is a tiny local app for making usable tabletop NPCs fast.

Download the ZIP, unzip it, double-click the launcher, and press `Generate NPC`.

Each NPC card gives you:

- what they want,
- what they know,
- what they ask the players for,
- what the players get,
- what goes wrong if ignored,
- a catch,
- a line to read aloud,
- and a trace showing why the chaos happened.

It is deterministic and local-first. It does not use AI APIs, accounts, cloud calls, Ollama, npm, or a build step.

This is a Windows-first public alpha. Python 3.10 or newer is required.

## Install Instructions

1. Download `NPCChaosBox-v0.1.0.zip`.
2. Unzip it somewhere easy, preferably on the D drive.
3. Double-click `START_NPCChaos_WINDOWS.bat`.
4. Press `Generate NPC`.

If Python is missing, install Python 3.10 or newer from python.org and tick `Add python.exe to PATH`.

## Public Alpha Note

This is an early public alpha. The main thing being tested is whether strangers can download, unzip, double-click, generate, save, and export without confusion.

# NPC Chaos Box v0.1.0 Public Alpha

![NPC Chaos Box Generate page](https://raw.githubusercontent.com/Martin123132/NPCChaosBox/v0.1.0/docs/assets/readme-generate.png)

NPC Chaos Box is a local-first NPC generator for tabletop games, writing, solo RPGs, and worldbuilding.

Press one button and get a character you can actually use at the table: what they want, what they know, what they ask the players to do, what happens if ignored, and the strange little twist that makes them memorable.

## Download And Start

1. Download `NPCChaosBox-v0.1.0.zip` from the assets below.
2. Unzip it somewhere easy, preferably on the D drive.
3. Double-click `START_NPCChaos_WINDOWS.bat`.
4. Your browser opens.
5. Press `Generate NPC`.

## What It Is

- Windows-first public alpha.
- No accounts.
- No AI API keys.
- No cloud calls.
- No Ollama.
- No npm or build step.
- Python 3.10 or newer required.

## What You Get

Each generated NPC includes:

- when to use them,
- how to open the scene,
- what they ask the players for,
- what the players get,
- the catch,
- what happens if ignored,
- a line to read aloud,
- and a deterministic chaos trace.

## Feedback

This is a public alpha. If something breaks, open the app's `Help` page, press `Copy Debug Report`, and paste it into the closest GitHub issue form:

https://github.com/Martin123132/NPCChaosBox/issues/new/choose

The first patch after this release should focus on launch friction, Python confusion, Windows warning confusion, save/export problems, and layout bugs rather than big new features.

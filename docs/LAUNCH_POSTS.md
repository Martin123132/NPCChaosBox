# Public Alpha Launch Posts

## Short Post

I made a local-first NPC generator for tabletop games.

No accounts, no AI keys, no cloud, no Ollama. Download the ZIP, unzip it, double-click the launcher, and press Generate NPC.

It makes characters with a usable scene hook: what they want, what they ask the players for, what the reward is, what the catch is, and what happens if ignored.

Public alpha is here:

https://github.com/Martin123132/NPCChaosBox/releases/tag/v0.1.0

## Slightly Longer Post

NPC Chaos Box is a Windows-first public alpha for making tabletop NPCs quickly.

It is local-first and deterministic. It does not use API keys, accounts, cloud calls, Ollama, npm, or a build step.

The point is not just to make a funny name. Each generated card gives you a scene you can actually run: open with this, ask the players for this, offer this reward, add this catch, and escalate this way if ignored.

Download ZIP, unzip, double-click, press Generate NPC:

https://github.com/Martin123132/NPCChaosBox/releases/tag/v0.1.0

If it breaks, the app has a Help page with a Copy Debug Report button.
